# Examples

This directory contains usage examples for the Profit2Pitfall toolkit.

## Available Examples

| Example | Description |
|---------|-------------|
| `network_evasion_example.py` | Demonstrates domain rotation and cloud loading detection |

## Running Examples

```bash
# Run network evasion detection example
python examples/network_evasion_example.py
```

## Prerequisites

Make sure you have installed all dependencies:

```bash
pip install -r requirements.txt
```

## Example Data

The examples use sample data files. You can replace them with your own data:

- Traffic capture files: Plain text files with domain names (one per line)
- MITM output files: HTTP response data captured by mitmproxy
